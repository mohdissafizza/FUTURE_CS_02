# Screenshots Folder

This folder contains screenshots used as evidence in the phishing email analysis.

## Files Included
- Safe, Suspicious and Phishing Email.png

## Description

### Safe Email
Example of a legitimate email with:
- Official domain  
- Personalized greeting  
- No urgency or threats  

### Suspicious Email
Example of a suspicious email with:
- Generic greeting  
- Unofficial domain  
- Suspicious link  

### Phishing Email
Example of a phishing email with:
- Fake sender  
- Malicious link  
- Urgency and threat tactics  

## Purpose
These screenshots visually demonstrate how different types of emails appear and help users identify phishing attempts.

## Note
All examples are created for educational and awareness purposes only
